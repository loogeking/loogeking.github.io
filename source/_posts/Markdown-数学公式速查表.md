#  Markdown 数学公式速查表（LaTeX 语法）

<hr>

## 基础语法

| 类型     | 写法       | 效果         |
| :------- | :--------- | :----------- |
| 行内公式 | `$公式$`   | 公式公式     |
| 块级公式 | `$$公式$$` | 独立一行居中 |

------

## 指数与上下标

| 描述           | 写法        | 效果           |
| :------------- | :---------- | :------------- |
| 上标           | `$a^b$`     | ab*a**b*       |
| 上标（多字符） | `$a^{bc}$`  | abc*a**b**c*   |
| 下标           | `$a_b$`     | ab*a**b*       |
| 下标（多字符） | `$a_{bc}$`  | abc*a**b**c*   |
| 上下标组合     | `$a_b^c$`   | abc*a**b**c*   |
| 指数函数       | `$e^x$`     | ex*e**x*       |
| 指数函数       | `$\exp(x)$` | exp⁡(x)exp(*x*) |

------

## 矩阵与行列式


```markdown
$$
\begin{pmatrix}
1 & 2 \\
3 & 4
\end{pmatrix}
$$
```

**行列式：**

```markdown
$$
\begin{vmatrix}
```

## 间距

| 写法          | 效果       | 说明           |
| :------------ | :--------- | :------------- |
| `$a\,b$`      | a b*a**b*  | 小间距         |
| `$a\;b$`      | a  b*a**b* | 中等间距       |
| `$a\quad b$`  | ab*a**b*   | 大间距         |
| `$a\qquad b$` | ab*a**b*   | 两倍大间距     |
| `$a\!b$`      | a ⁣b*a**b*  | 负间距（紧贴） |

## 综合示例

**指数与对数组合：**

```markdown
$e^{\ln x} = x$
$\log_{2}(2^n) = n$
```

**复杂公式：**

```markdown
$$
f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x-\mu}{\sigma}\right)^2}
$$
```

**多行公式：**

```markdown
$$
\begin{aligned}
e^{i\pi} &= \cos\pi + i\sin\pi \\
&= -1 + 0 \\
&= -1
\end{aligned}
$$
```
